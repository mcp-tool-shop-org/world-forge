# @world-forge/renderer-2d

[वर्ल्ड फोर्ज](https://github.com/mcp-tool-shop-org/world-forge) के वर्ल्ड मैप्स के लिए PixiJS-आधारित 2D रेंडरर।

## इंस्टॉलेशन

```bash
npm install @world-forge/renderer-2d
```

## घटक

- **वर्ल्डव्यूपोर्ट:** पैन, ज़ूम और ग्रिड ओवरले के साथ PixiJS एप्लिकेशन रैपर।
- **ज़ोनओवरलेरेंडरर:** ज़ोन की सीमाओं को जिले के रंगों के साथ दर्शाना, चयन/होवर सुविधा।
- **कनेक्शनरेंडरर:** ज़ोन के बीच रेखाएं, एक-तरफ़ा कनेक्शन के लिए तीर, और सशर्त कनेक्शन के लिए बिंदीदार रेखाएं।
- **एंटिटीरेंडरर:** भूमिका-आधारित एंटिटी आइकन (एनपीसी, दुश्मन, व्यापारी, बॉस)।
- **टाइललेयररेंडरर:** टैग-आधारित रंगों के साथ z-क्रमित टाइल लेयर।
- **मिनिमैपरेंडरर:** व्यूपोर्ट इंडिकेटर के साथ स्केल किया गया अवलोकन।
