# @world-forge/editor

React 19 वेब एप्लिकेशन, जो [वर्ल्ड फोर्ज](https://github.com/mcp-tool-shop-org/world-forge) के लिए बनाया गया है — यह एक 2डी दुनिया निर्माण स्टूडियो है, जो एआई आरपीजी इंजन के लिए है।

## विशेषताएं

- **कैनवास:** 2डी ग्रिड पर ज़ोन बनाएं, कनेक्शन बनाएं, एंटिटीज, लैंडमार्क और मुठभेड़ रखें, साथ ही पैन/ज़ूम व्यूपोर्ट का उपयोग करें।
- **मल्टी-सेलेक्ट:** ज़ोन, एंटिटीज, लैंडमार्क, स्पॉन और मुठभेड़ को क्लिक करके, शिफ्ट-क्लिक करके या बॉक्स-सेलेक्ट करके चुनें; चयनित ऑब्जेक्ट्स को ड्रैग करके ले जाएं और 'अंडू' (पूर्ववत) सुविधा का उपयोग करें।
- **मुठभेड़ निर्माण:** ज़ोन पर मुठभेड़ एंकर रखें, जिसमें टाइप-आधारित कैनवास मार्कर (बॉस, घात, गश्त) हों, और जिन्हें संपादित किया जा सकता है (शत्रु आईडी, संभावना, कूलडाउन, टैग)।
- **जिला संपादन:** विस्तारित जिला पैनल, जिसमें मेट्रिक्स स्लाइडर, टैग, नियंत्रित गुट, अर्थव्यवस्था प्रोफाइल, गुट की उपस्थिति प्रबंधन और दबाव हॉटस्पॉट संपादन शामिल हैं।
- **बैच संपादन:** कई ज़ोन चुनने पर, बैच में जिला असाइनमेंट, बैच में टैग जोड़ना और बैच में डिलीट करना।
- **प्रीसेट सिस्टम:** क्षेत्र और मुठभेड़ प्रीसेट, जिन्हें मर्ज/ओवरराइट किया जा सकता है; 4 अंतर्निहित क्षेत्र प्रीसेट, 3 अंतर्निहित मुठभेड़ प्रीसेट, कस्टम प्रीसेट CRUD (बनाना, पढ़ना, अपडेट करना, डिलीट करना) और localStorage में डेटा सुरक्षित रखने की सुविधा।
- **कीबोर्ड शॉर्टकट:** केंद्रीकृत हॉटकी रजिस्ट्री, जिसमें 13 शॉर्टकट हैं: Escape, Ctrl+A, Ctrl+D, Ctrl+K, Delete, तीर कुंजियों से थोड़ा-थोड़ा खिसकाना, Enter (विवरण खोलना), P (प्रीसेट लागू करना), Shift+P (प्रीसेट सहेजना)।
- **डबल-क्लिक:** किसी भी कैनवास ऑब्जेक्ट पर डबल-क्लिक करके उसे चुनें, 'मैप' टैब पर जाएं और व्यूपोर्ट को केंद्रित करें।
- **स्पीड पैनल:** कैनवास पर डबल-राइट-क्लिक करके एक फ्लोटिंग कमांड पैलेट खोलें, जिसमें संदर्भ-जागरूक क्रियाएं, पिन करने योग्य पसंदीदा (संपादन मोड में पुन: व्यवस्थित करने योग्य), हाल की क्रियाएं, कस्टम समूह, हल्के मैक्रो (स्टेप एडिटर के साथ), खोज फ़िल्टरिंग और कीबोर्ड नेविगेशन शामिल हैं।
- **वर्कस्पेस टैब:** मैप, प्लेयर, बिल्ड, ट्री, डायलॉग, ऑब्जेक्ट, प्रीसेट, एसेट, इश्यू, गाइड।
- **एसेट लाइब्रेरी:** पोर्ट्रेट, स्प्राइट, बैकग्राउंड, आइकन और टाइलसेट को प्रबंधित करें, जिनमें प्रत्येक प्रकार के लिए विशिष्ट सेटिंग्स हैं।
- **अंडू/रीडू:** Zustand के माध्यम से 10 स्तरों तक इतिहास को पूर्ववत/पुन: करें।
- **इम्पोर्ट/एक्सपोर्ट:** राउंड-ट्रिप फ़िडेलिटी रिपोर्टिंग, सिमेंटिक डिफ़ ट्रैकिंग।
- **वैलिडेशन:** 54 संरचनात्मक जांच, जिनमें क्लिक करके समस्या को हाइलाइट करने की सुविधा है।
- **टेम्प्लेट:** त्वरित शुरुआत के लिए विभिन्न शैलियों के शुरुआती बिंदु और नमूना दुनिया।

## शुरुआत कैसे करें

```bash
npm install
npm run dev --workspace=packages/editor
```

एडिटर लॉन्च करने के लिए `http://localhost:5173` खोलें।
